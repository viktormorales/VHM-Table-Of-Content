Show a Table Of Content (TOC) in your posts.

== Description ==

Table Of Content (TOC) are usefull when your posts are quite long. This allows users to find an specific section of your post using the [vhm-toc] shortcode.

== Installation ==

1. Upload `vhm-toc` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the plugin settings `Settings > VHM Table Of Content`
1. In your posts add the shortcode [vhm-toc] where you want to display the table of content.
